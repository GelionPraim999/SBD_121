Переменная (variable) - именованная область памяти, содержимое которой может изменяться 
в процессе выполнения программы.
Прежде чем использовать переменную, ее нужно объявить (создать).
			
			Синтаксис объявления переменной:
					type name;
type - тип объявляемой переменной. 
Он определяет три вещи:
	- сколько памяти переменная будет заниамть;
	- какие значения она сможет принимать;
	- какие операции над ней можно выполнять;
1. Логические типы: bool (Boolean). 
				true	false
2. Символьные типы: char (Character - Символ) 1 Byte
   ASCII - American Standard Code for Information Interchacge.
   https://github.com/okovtun/BD_011/blob/master/NumericSystems.pdf
3. Числовые типы. Делятся на целочисленные и вещественные.
   Беззнаковые (unsigned) - могут хранить лишь положительные целые числа;
   Знаковые (signed) - могут хранить как положительные, так и отрицательные целые числа

   long 
   sizeof(type);
